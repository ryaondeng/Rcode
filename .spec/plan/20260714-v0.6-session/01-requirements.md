# v0.6 会话管理 — 需求文档

> 状态：待确认

## 1. 背景与目标

### 1.1 背景

v0.5 已实现 Trace 追踪，但会话状态无法持久化。需要实现会话管理，支持多轮对话和会话恢复。

### 1.2 目标

- 实现 Session 数据结构
- 实现 SessionStore 文件持久化
- 实现 SessionManager 业务管理
- 支持会话创建、发送消息、关闭

## 2. 用户场景

| 场景 | 用户 | 触发条件 | 期望结果 |
|------|------|----------|----------|
| 多轮对话 | 开发者 | 连续提问 | 保持上下文 |
| 会话恢复 | 开发者 | 重启后继续 | 恢复之前的会话 |
| 会话管理 | 开发者 | 管理会话 | 创建、关闭会话 |

## 3. 功能范围

### 3.1 做什么（In Scope）

- [ ] Session 数据结构
- [ ] SessionStore 文件持久化
- [ ] SessionManager 业务管理
- [ ] 会话状态机（active/waiting_for_input/closed）

### 3.2 不做什么（Out of Scope）

- Session 并发控制（简化版）
- Session 搜索和过滤
- Session 导出
