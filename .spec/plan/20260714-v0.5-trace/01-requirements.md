## 1. 背景与目标

### 1.1 背景

v0.4 已实现事件系统，但事件只在内存中，无法持久化和回放。需要实现 Trace 系统，记录完整的执行时间线，支持事后分析和调试。

### 1.2 目标

- 实现 TraceRecord 统一数据结构
- 实现 TraceWriter 异步写入 JSONL
- 实现 TracingProvider 装饰器追踪 LLM
- 支持 IPC/Event/LLM 三层追踪

## 2. 用户场景

| 场景 | 用户 | 触发条件 | 期望结果 |
|------|------|----------|----------|
| 调试分析 | 开发者 | Agent 行为异常 | 回放 trace 定位问题 |
| 性能分析 | 开发者 | 需要优化性能 | 查看 LLM 调用延迟 |
| 审计追踪 | 管理员 | 需要操作记录 | 查看完整执行历史 |

## 3. 功能范围
