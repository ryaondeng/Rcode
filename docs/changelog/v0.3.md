# v0.3 工具扩展

> 发布日期：2026-07-12

## 概述

扩展内置工具，提升 Agent 的文件操作能力。

## 新增功能

- **safe_path**：路径安全校验，防止路径逃逸
- **read_file**：读取文件内容
- **write_file**：写入文件
- **edit_file**：精确文本替换
- **list_dir**：列出目录内容
- **glob**：文件模式匹配

## 工具清单

| 工具 | 功能 | 安全机制 |
|------|------|----------|
| bash | 执行命令 | 超时保护 |
| read_file | 读取文件 | safe_path + 512KB 限制 |
| write_file | 写入文件 | safe_path + 1MB 限制 |
| edit_file | 文本替换 | safe_path |
| list_dir | 目录列表 | safe_path + 深度限制 |
| glob | 文件匹配 | safe_path + 结果校验 |
