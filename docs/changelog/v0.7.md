# v0.7 上下文压缩

> 发布日期：2026-07-14

## 概述

实现上下文压缩，支持长对话场景下的 token 管理。

## 新增功能

- **CompactionConfig**：压缩配置
- **truncate_tool_results**：超长 tool_result 截断
- **Compactor**：LLM 驱动的上下文压缩器

## 四层压缩管线

| 层 | 函数 | 成本 | 触发条件 |
|---|---|---|---|
| L3 | tool_result_budget() | 0 API | tool_result > 200KB |
| L1 | snip_compact() | 0 API | 消息数 > 50 |
| L2 | micro_compact() | 0 API | tool_result 总数 > 3 |
| L4 | compact_history() | 1 API | token > 50000 |
