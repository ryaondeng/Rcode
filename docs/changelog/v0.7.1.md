# v0.7.1 Session 接线重构

> 发布日期：2026-07-16

## 概述

将 SessionManager 接入 AgentRunner 主链路，支持多轮对话和 resume 能力。

## 变更

### AgentRunner

- `run()` 签名变更：支持 `run(session_id, user_input)` 模式
- 集成 SessionManager，自动加载/创建 session
- 发布 `session.attached` / `session.detached` 事件

### SessionManager

- 新增 `load()` 方法：加载已有 session
- 新增 `load_context()` 方法：从 session 构建 ExecutionContext
- 新增 `save()` 方法：保存消息到 session

### 事件类型

- 新增 `SessionAttachedEvent`：session 连接事件
- 新增 `SessionDetachedEvent`：session 断开事件

## 向后兼容

- 旧的 `run(goal)` 模式仍可用（创建一次性 session）

## 测试

- 单元测试：110 个（全部通过）
- 集成测试：跳过（纯模块重构）
