# v0.7.5 配置系统统一

> 发布日期：2026-07-16

## 概述

将配置统一到"四层优先级 + pydantic 校验 + 未知 key 硬拒"的配置模型。

## 变更

### 配置模型

- 新增 LLMConfig、SessionConfig、CompactConfig、TraceConfig、LoggingConfig
- RcodeConfig 支持 pydantic 校验，未知 key 硬拒

### 配置加载

- 支持四层优先级：默认值 → 全局 → 本地 → 环境变量/CLI
- 支持环境变量 RCODE_* 覆盖

### CLI

- 新增 `config show` 命令

## 向后兼容

- 默认值与现有行为一致

## 测试

- 单元测试：153 个（全部通过）
- 覆盖率：78%
