# v0.7.2 Compactor 接线重构

> 发布日期：2026-07-16

## 概述

将 Compactor 接入 AgentLoop 主链路，实现上下文自动压缩。

## 变更

### AgentLoop

- 新增 `compactor` 和 `compact_threshold` 参数
- 每步开始前检查水位（TokenBudget.check）
- 超水位时触发压缩（_do_compact）
- 压缩失败时保持原 messages，不中断主循环

### TokenBudget

- 新增 `check()` 方法：检查是否需要压缩

### ExecutionContext

- 新增 `replace_history()` 方法：用摘要替换历史消息

### 事件类型

- 新增 `CompactTriggeredEvent`：压缩触发事件
- 新增 `CompactFinishedEvent`：压缩完成事件
- 新增 `CompactFailedEvent`：压缩失败事件

### Runner

- 传入 Compactor 和 compact_threshold

## 向后兼容

- 默认 `compact_threshold=0.0`，禁用自动压缩

## 测试

- 单元测试：138 个（全部通过）
- 覆盖率：79%
