# v0.5 Trace 追踪

> 发布日期：2026-07-13

## 概述

实现 Trace 追踪系统，记录 Agent 执行过程中的 LLM 调用和事件。

## 新增功能

- **TraceRecord**：统一的追踪记录模型
- **TraceWriter**：异步 JSONL 写入器
- **TracingProvider**：包裹 LLM Provider，自动记录调用

## Trace 结构

```json
{
  "ts": "时间戳",
  "direction": "数据流向",
  "layer": "层级（llm/event/ipc）",
  "kind": "事件类型",
  "data": {}
}
```

## 层级说明

| 层级 | 用途 |
|------|------|
| llm | LLM API 调用 |
| event | EventBus 事件 |
| ipc | 进程间通信 |
