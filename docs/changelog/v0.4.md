# v0.4 事件流

> 发布日期：2026-07-13

## 概述

实现事件系统，将 Agent 执行过程通过 EventBus 实时外化。

## 新增功能

- **EventBus**：发布-订阅事件总线
- **Event Types**：8 种事件类型定义
- **EventWriter**：JSONL 事件持久化
- **Agent Loop 事件集成**：每轮发布事件

## 事件类型

| 事件 | 用途 |
|------|------|
| run.started/finished | Run 生命周期 |
| step.started/finished | Step 生命周期 |
| tool.call_started/finished | 工具调用 |
| llm.call_started/finished | LLM 监控 |
