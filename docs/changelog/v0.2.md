# v0.2 Agent 最小闭环

> 发布日期：2026-07-12

## 概述

实现最小可运行的 Agent Loop，支持 LLM 调用和工具执行。

## 新增功能

- **Agent Loop**：ReAct 循环（Think → Act → Observe）
- **LLM Provider**：Anthropic API 接口
- **Tool System**：工具注册表和调用机制
- **ExecutionContext**：执行上下文管理
- **AgentRunner**：组装所有依赖
- **Bash Tool**：内置 bash 工具
- **rcode run 命令**：执行 Agent 任务

## 命令

```bash
rcode run --goal "创建一个 hello.py 文件"  # 执行 Agent 任务
```
