# v0.6 会话管理

> 发布日期：2026-07-13

## 概述

实现会话管理系统，支持多轮对话和会话持久化。

## 新增功能

- **Session**：会话数据结构
- **SessionStore**：文件系统存储
- **SessionManager**：会话业务管理

## 目录结构

```
{session_id}/
├── meta.json        # 会话元数据
├── thread.jsonl     # 对话历史
└── notes.md         # 会话笔记
```
