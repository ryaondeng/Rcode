# v0.1 骨架与协议

> 发布日期：2026-07-12

## 概述

建立双进程架构，CLI 和 Core 通过 JSON-RPC 2.0 + NDJSON 完成 IPC 通信。

## 新增功能

- **双进程架构**：rcode-core 守护进程 + rcode CLI 客户端
- **JSON-RPC 2.0 协议**：标准化的请求/响应/错误格式
- **TCP 传输层**：NDJSON 行协议，便于流式读取
- **ping/pong 命令**：验证通信链路
- **配置管理**：四级优先级（默认值 → 全局 TOML → 项目 TOML → .env → 环境变量）
- **端口占用检测**：启动时检测端口冲突
- **优雅关闭**：Ctrl+C 信号处理

## 命令

```bash
rcode-core          # 启动 Core 守护进程
rcode ping          # 测试连接
rcode --version     # 查看版本
```
