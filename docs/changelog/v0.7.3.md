# v0.7.3 事件订阅装配解耦

> 发布日期：2026-07-16

## 概述

将事件订阅者从 AgentRunner 硬编码中解耦，支持按配置灵活装配。

## 变更

### 新增模块

- `events/registry.py`：SubscriberRegistry 和 build_default_subscribers
- `cli/printer.py`：ConsoleSubscriber（从 runner.py 挪出）

### Runner

- 移除硬编码的 _EventPrinter
- 使用 SubscriberRegistry 按配置订阅事件

### 错误处理

- 订阅者异常不影响其他订阅者（错误隔离）

## 向后兼容

- 终端输出效果与重构前一致

## 测试

- 单元测试：143 个（全部通过）
- 覆盖率：80%
