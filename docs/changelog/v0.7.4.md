# v0.7.4 CLI ↔ Core IPC 化（骨架）

> 发布日期：2026-07-16

## 概述

实现 CLIENT / CORE 双进程骨架，CLI 通过 JSON-RPC 与 Core 通信。

## 变更

### 新增模块

- `ipc/client.py`：IpcClient，JSON-RPC 客户端

### CLI

- 新增 `core` 命令组：start/stop/status
- `run` 命令支持 `--local` 参数

### SocketServer

- 支持 core.run 异步执行

## 向后兼容

- `--local` 保留同进程行为

- 单元测试：149 个（全部通过）
- 覆盖率：79%
