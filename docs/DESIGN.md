# Rcode 设计文档

> **定位**：本地 AI Agent 系统，参考 Claude Code 核心架构实现

---

## 设计理念

```
Agent = Model (LLM) + Harness (运行环境)
```

Rcode 实现的是 **Harness**：工具、权限、上下文管理、会话续航。

---

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                      用户界面层                              │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐                  │
│  │   CLI    │  │   TUI    │  │   Web    │  (未来扩展)       │
│  └────┬─────┘  └────┬─────┘  └────┬─────┘                  │
│       └──────────────┼──────────────┘                        │
│              JSON-RPC 2.0 + NDJSON                           │
├─────────────────────────────────────────────────────────────┤
│                    Core 守护进程                              │
│  ┌─────────────────────────────────────────────────────┐    │
│  │                    AgentRunner                       │    │
│  │              AgentLoop (Think → Act → Observe)      │    │
│  └─────────────────────────────────────────────────────┘    │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐       │
│  │   Tools  │ │Permission│ │  Events  │ │ Session  │       │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘       │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐       │
│  │  Memory  │ │ Compact  │ │ Subagent │ │   MCP    │       │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘       │
└─────────────────────────────────────────────────────────────┘
```
